## What it is

UCann is the structural answer to the cannabis-tech problem: every operator-facing platform extracts the operator's data, sells it back as "insights," and locks the operator into the platform with vendor-managed pricing. UCann inverts that. The operator owns the data, the AI runs locally on Apple Silicon, the platform never sees the recipe.

Five-mechanism business trust stack: data equity (operator owns), local inference (no cloud extraction), exclusive metro license windows (420-day cohort exclusivity per market), founder license access path, public continuity pledge. Each mechanism reinforces the others.

## Pronunciation

UCann is pronounced "you-CANN" (two syllables, emphasis on the second). The double-N matters: it signals the cannabis-industry suffix and distinguishes the brand from the verb "you can." Visual artifacts render the brand as UCann (capital U, capital C, double-n).

---

## Stack

Built with TypeScript, Node, Stripe, local MLX inference for the operator-facing AI layer. 420-day metro license cohort structure (per-market exclusivity windows that customers buy and renew). Public continuity pledge with NCC Group source escrow. Data stays with the operator; UCann is the engine, not the warehouse.
